Введение в Frontend Разработку

Frontend-разработка отвечает за создание пользовательского интерфейса пользователькой части веб-сайт или веб-приложение. Она включает в себя работу с HTML, CSS и JavaScript».

Языки Frontend Разработки

Основными языками Frontend разработки являются:

  • HTML (HyperText Markup Language) - используется для структурирования содержимого веб-страница
  • CSS (Cascading Style Sheets) используется для стилизовать веб-страницы и создать ее внешний вида
  • JavaScript используется для добавления интерактивности в веб-страницу, такой как анимации, обработка событий и динамическое обновление содержимого

Инструменты Frontend Разработки

  1. Visual Studio Code — мощный редактор кода с поддержкой множества языков и плагинов
  2. React — библиотека для создания пользовательских интерфейсов
  3. Sass — репроцессор CSS с расширенными возможностями
  4. Webpack — Сборщик модулей для JavaScript приложений

Статьи

Что такое frontend-разработка: создание интерфейса приложений, включающего кнопки, баннеры, формы и другие элементы, с использованием HTML, CSS и JavaScript.

Основная задача frontend-разработчика: превращение рабочей страницы в красивый макет, созданный дизайнерами и UX-аналитиками, и реализация логики поведения элементов страницы.

Необходимые знания и навыки для frontend-разработчика: владение тремя основными языками (HTML, CSS, JavaScript), системой контроля версий (GitHub/GitLab), протоколом HTTP и REST API, а также популярными фреймворками (React, Angular, Vue).
